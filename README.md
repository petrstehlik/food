# REST API
Requirements
```
beautifulsoup4==4.5.1
click==6.6
Flask==0.11.1
itsdangerous==0.24
Jinja2==2.8
MarkupSafe==0.23
pycurl==7.43.0
requests==2.11.1
Werkzeug==0.11.11
```

It is better to make virtualenv (`virtualenv food`) and then `pip install -r requirements.txt`.
Activate virtualenv and run api.py.

# Installation
`npm install`

# How to run
`npm start`
